Blockchains have varying confirmation depths to consider blocks as final.

| Chain    | `chainId` | Confirmations required | Confirmation time |
| -------- | --------- | ---------------------- | ----------------- |
| Bitcoin  | 0         | 6                      | ≈60 mins          |
| Dogecoin | 2         | 60                     | ≈60 mins          |
| XRPL     | 3         | 3                      | ≈12 seconds       |
